# **App Name**: CalcZilla

## Core Features:

- Number Input: Allow users to input numbers for calculations.
- Operator Input: Enable users to input basic arithmetic operators (+, -, *, /).
- Calculation Display: Display the current calculation and result dynamically.
- Clear Function: Implement a clear button to reset the calculation.
- Memory Functions: Implement memory store, recall, add and subtract functions.

## Style Guidelines:

- Background color: Dark gray (#121212) for a modern dark theme.
- Primary color: Vibrant green (#32CD32) for numbers.
- Accent color: White (#FFFFFF) for operators to ensure clarity against the dark background.
- Font: 'Inter', a grotesque-style sans-serif, for both numerals and operators. This provides a modern and clean interface.
- Organize number and operator buttons in a standard calculator layout for intuitive use.
- Implement a subtle button press animation for visual feedback.
